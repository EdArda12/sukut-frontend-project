# sukut-frontend-project
Grafik tasarımcılar tarafından hazırlanan özgün arayüz tasarımları referans alınarak, HTML, CSS ve JavaScript kullanımıyla web ortamına en yakın şekilde aktarılmıştır. Çok sayfalı yapı, responsive tasarım ve modern UI/UX prensipleriyle geliştirilen frontend odaklı bir projedir.
