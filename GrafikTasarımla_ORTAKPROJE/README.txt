SÜKÛT – Uyku & Zihin Odaklı Web Sitesi
README / ÇALIŞTIRMA TALİMATLARI

1. PROJEYİ AÇMA

Bu proje, zip dosyası açıldıktan sonra tek bir klasör içerisinde çalışacak şekilde hazırlanmıştır.
Klasör içinde birden fazla HTML, CSS, JavaScript dosyası ve görsel klasörleri bulunmaktadır.

--------------------------------------------------
2. VISUAL STUDIO CODE İLE AÇMA (ÖNERİLEN YÖNTEM)
--------------------------------------------------

Eğer bilgisayarınızda Visual Studio Code (VS Code) yüklüyse:

1. Proje klasörünü Visual Studio Code ile açınız.
2. Sağ altta bulunan “Go Live” butonuna tıklayınız.
3. Web sitesi varsayılan tarayıcıda otomatik olarak açılacaktır.

Bu yöntemle siteyi açmanız önerilir.
Çünkü:
- Sayfalar arası bağlantılar daha sağlıklı çalışır
- CSS ve görsel yollarında oluşabilecek hatalar minimuma iner
- Proje, gerçek bir web ortamına en yakın şekilde çalışır

Bu işlemin nasıl yapıldığı,
“WEB SİTESİ VİDEOSU.mkv” adlı ekran kaydı dosyasında adım adım gösterilmiştir.

--------------------------------------------------
3. VISUAL STUDIO CODE OLMADAN AÇMA
--------------------------------------------------

Eğer bilgisayarınızda Visual Studio Code yüklü değilse:

1. Proje klasörü içinden “index.html” dosyasına çift tıklayınız.
2. Web sitesi tarayıcıda açılacaktır.
3. Üst menüden (Ana Sayfa, Hakkımızda, İçeriğimiz, Ürünlerimiz, Duyurular, İletişim)
   sayfalar arasında geçiş yapabilirsiniz.

Bu yöntem de çalışır; ancak VS Code + Go Live yöntemi önerilmektedir.

--------------------------------------------------
4. PROJE YAPISI
--------------------------------------------------

- index.html            → Ana Sayfa
- hakkimizda.html       → Hakkımızda
- icerigimiz.html       → İçeriğimiz
- urunlerimiz.html      → Ürünlerimiz
- duyurular.html        → Duyurular
- iletisim.html         → İletişim

- style.css             → Ana stil dosyası
- overrides.css         → Sayfa bazlı ek ve override stiller
- script.js             → JavaScript dosyası

- images/               → Web sitesinde kullanılan görseller
- fonts/                → Tasarımda kullanılan fontlar
